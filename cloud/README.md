# 云平台

## 主流云服务商

### 国内云
- **阿里云** (Alibaba Cloud)
- **腾讯云** (Tencent Cloud)
- **华为云** (Huawei Cloud)

### 国际云
- **AWS** (Amazon Web Services)
- **Google Cloud**
- **Azure** (Microsoft Azure)

## 核心服务

### 计算
- ECS / EC2 (虚拟机)
- 容器服务 (ACK, EKS)
- Serverless (函数计算)

### 存储
- OSS / S3 (对象存储)
- 块存储
- 文件存储 (NAS)

### 网络
- VPC (虚拟私有云)
- 负载均衡 (SLB, ELB)
- CDN (内容分发)

### 数据库
- RDS (关系型数据库)
- NoSQL (MongoDB, Redis)
- 数据仓库

## 多云策略

### 优势
- 避免厂商锁定
- 提高可用性
- 成本优化

### 挑战
- 运维复杂度增加
- 数据同步
- 统一管理
