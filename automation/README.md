## 基础设施即代码 (IaC)

### 原则
1. **版本控制** - 所有配置代码化
2. **幂等性** - 重复执行结果一致
3. **可测试** - 配置变更前测试
4. **可回滚** - 失败时快速恢复

### Terraform 示例
```hcl
resource "aws_instance" "web" {
  ami           = "ami-12345678"
  instance_type = "t2.micro"
  
  tags = {
    Name = "WebServer"
  }
}
```
